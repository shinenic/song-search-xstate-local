## Song search xstate
主要針對手機使用者，協助需要快速查找`卓著樂譜曲目`的使用者，能方便快速尋找所需內容。

[Demo Link](https://shinenic.github.io/song-search-xstate-local/)

## 使用技術

* React Hooks
	+ 使用`custom hooks`整理Component，讓`畫面`跟`邏輯`分離
* XState
	+ 管理`input`輸入到處理`api非同步`的各種狀態 (可於底下state chart預覽)
* Infinite scroll
* Debounce Search
* SCSS
